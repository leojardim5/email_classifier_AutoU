# Pasta Public

Esta pasta contém arquivos estáticos que serão servidos diretamente pelo Next.js.

## Logo AutoU

Para adicionar o logo da AutoU, coloque o arquivo `logo-autou.png` nesta pasta (`frontend/public/logo-autou.png`).

O arquivo deve ter:
- Fundo transparente (PNG com alpha)
- Resolução alta recomendada (mínimo 384x128 pixels)
- Nome do arquivo: `logo-autou.png`

O componente `AutoULogo` já está configurado para usar este arquivo automaticamente.
